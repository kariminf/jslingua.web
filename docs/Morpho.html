    static, readonly Feature :Object
    morpho.js, line 315

    This is a map to different morphology features:
    • POS: It returns Morpho.PoS
    • Tense: It returns Morpho.Tense
    • Aspect: It returns Morpho.Aspect
    • Mood: It returns Morpho.Mood
    • Voice: It returns Morpho.Voice
    • Number: It returns Morpho.GNumber
    • Person: It returns Morpho.Person
    • Gender: It returns Morpho.Gender
    We can access these features either by:
    Morpho.Feature.feature_name
    Or:
    Morpho.Feature["feature_name"]

    static getOptName(optLabel, opts) → {String}
    morpho.js, line 593

    Returns the name of a conjugation parameter (Pronoun, Negation, Voice) given some options
    Parameters:
    Name Type Description
    optLabel String can be: Pronoun, Negation, Voice
    opts Object The parameters
    Returns:
    String - The label of this parameter in the current language
    Example
    var opts = {
     person: "first", // Morpho.Feature.Person.F
     number: "singular" // Morpho.Feature.Number.S
   };
   var I = getOptName("Pronoun", opts);
// In English, it will give: "I"
// In Arabic, it will give: "أنا"
